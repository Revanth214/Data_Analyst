# Chocolate Sales Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze chocolate sales performance using 20,000+ records.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## Key Metrics
- Total Sales: $141M
- Total Profit: $81M
- Shipment Count: 25K
- Profit Percentage: 57.32%

## Dashboard Features
- Sales Trend Analysis
- Product Performance Analysis
- Shipment Distribution
- Geographic Analysis
- Profitability Analysis
- Interactive Date Filters

## Business Insights
- Identified top-performing products.
- Tracked profit trends over time.
- Compared regional sales performance.
- Highlighted high-profit and low-profit products.
